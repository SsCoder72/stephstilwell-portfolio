# Stephanie Stilwell — Web Developer Portfolio

My personal portfolio website, built from scratch with HTML, CSS, and vanilla JavaScript. It brings together my projects, skills, and a little bit of who I am — designed around a clean, code-inspired aesthetic.

## About

I'm a self-directed web developer based in San Francisco with a strong administrative and customer-service background and a creative eye for design. This site is both my portfolio and a hands-on project for practicing modern front-end techniques — every page was hand-coded and styled by me.

## Features

- **Responsive, multi-page layout** that adapts from desktop down to mobile
- **Animated typing "code window"** on the home page, built with JavaScript
- **iMac screen overlay** on the About page — the bio sits inside the monitor using absolute positioning and container-query units so the text scales with the image
- **Interactive skills dropdowns** on the Projects page
- **Terminal-style contact card** with clickable email and phone links
- **Custom theme** — terminal greens, monospace type, and gradient nav buttons

## Pages

- **Home** — intro and animated code window
- **About** — my background and a more personal hello
